# VimX 

VimX is an UNIX text editor for programmers based on Vim.

## Compiling

    cd src
    ./configure --with-features=huge
    make
    make install

For details, see the `INSTALL` file in the `src` directory.

## Features

* Removed unnecessary readme files.
* Removed windows support.
